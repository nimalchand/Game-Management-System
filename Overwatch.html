Overwatch 2 System Requirements

Overwatch 2

Season 10: Venture Forth

Before joining the fight for the future, make sure your system meets the requirements below for optimal performance in Overwatch 2's team-based battles.

Processor

Speed:3.0 GHz

Memory & Storage

RAM:6 GB
Storage:50 GB (HDD)

Graphics & OS

GPU:NVIDIA GTX 600 / AMD HD 7000
OS:Windows 10 (64-bit)

Processor

Speed:3.5 GHz

Memory & Storage

RAM:8 GB

Graphics & OS

GPU:NVIDIA GTX 1060 / AMD RX 580
OS:Windows 10/11 (64-bit)

Speed:3.7 GHz

Memory & Storage

RAM:16 GB

Graphics & OS

GPU:RTX 2070 / RX 5700 XT
